Web Dev

Membuat Tampilan Minimalis dengan Tailwind CSS dan Dark Mode

Tailwind CSS adalah kerangka kerja CSS berbasis utilitas yang sangat membantu developer untuk membangun desain antarmuka secara cepat tanpa harus menulis kode CSS yang panjang dari nol.

Dalam artikel ini, kita membahas bagaimana memanfaatkan kelas-kelas bawaan Tailwind untuk membuat elemen responsif, mulai dari tampilan layar kecil di ponsel hingga layar besar di desktop.

Fitur mode gelap (dark mode) juga menjadi poin penting agar kenyamanan mata pengguna terjaga saat membaca konten di malam hari. Cukup dengan menambahkan kelas pendukung dan sedikit skrip JavaScript sederhana, fitur sakelar tema dapat berfungsi dengan mulus.

Trading

Strategi Membaca Tren Pergerakan XAU/USD di Sesi London

Sesi perdagangan London sering kali menjadi salah satu waktu paling aktif dan volatil bagi instrumen XAU/USD (Emas terhadap Dolar Amerika Serikat). Pergerakan harga di awal sesi ini sering menentukan arah tren harian.

Manajemen risiko adalah kunci utama sebelum memutuskan untuk membuka posisi. Selalu tentukan batasan risiko (*stop loss*) yang jelas dan sesuaikan ukuran lot dengan ketahanan modal akun Anda.

Catatan harian ini berfungsi untuk mengevaluasi kembali keputusan analisis teknikal yang telah diambil, baik saat posisi mengalami keuntungan maupun kerugian.

Web Dev

Memanfaatkan GitHub Pages untuk Hosting Website Gratis

GitHub Pages adalah fitur luar biasa yang disediakan oleh GitHub bagi para pengembang untuk meng-host situs web statis langsung dari repositori mereka secara cuma-cuma.

Prosesnya sangat mudah, Anda cukup mengunggah file HTML ke branch utama, mengaktifkan pengaturannya di menu Pages, dan website Anda langsung bisa diakses secara global.

Layanan ini sangat cocok bagi pemula yang ingin membangun portofolio, dokumentasi proyek, atau blog pribadi tanpa harus pusing memikirkan biaya sewa server bulanan.

Trading

Pentingnya Psikologi Trading dan Disiplin Risiko

Banyak trader pemula mengira bahwa kunci sukses di pasar finansial sepenuhnya terletak pada indikator teknikal yang rumit. Padahal, faktor psikologi dan emosi memegang porsi yang sangat besar.

Rasa serakah saat profit atau panik saat mengalami kerugian (*loss*) sering kali merusak rencana perdagangan yang sudah disusun rapi sebelumnya.

Disiplin dalam menerapkan manajemen risiko yang ketat adalah satu-satunya perisai agar akun trading bisa bertahan dalam jangka panjang di tengah ketidakpastian pasar.
